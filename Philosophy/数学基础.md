# 逻辑主义和形式主义

（弗雷格等人的）逻辑主义和今天我们所熟悉的那种形式主义的不同在于前者将所有数学对象约化为逻辑——请注意是无可争议的逻辑而不是集合论。
作为一个例子，逻辑主义视角下的函数最自然地应该使用摹状词来定义。
类似的，命题
$$
\# x P(x) = 2
$$
应该理解为
$$
\exist x \exist y (P(x) \wedge P(y) \wedge x \neq y \wedge \forall z (P(z) \rightarrow x=z \lor y=z ) )
$$

上面这种风格的定义不需要断言数学对象存在。的确上面的命题中出现了$\exist$但是我们从来不把形如$\exist x P(x)$的命题当作公理。
相反，上面的定义将涉及这个数学对象的命题转化成不涉及这个数学对象的命题。

弗雷格承认有集合存在。从传统逻辑学的角度而言，这不是一个违背了逻辑主义基本思路的做法，因为一个集合可以理解为一个概念（谓词）的外延，而逻辑的确讨论外延。
承认集合的存在性有一个好处，就是这样$\# x P(x)$很好定义（如可以定义为所有满足“二性”的东西——即上面的命题——的集合）。
当然，我们今天知道他的朴素的定义会导致罗素悖论。
对这个问题的最直截了当的修正可能是罗素的分支类型论（不过罗素的类型论对集合的定义和摹状词很相似，是“翻译”式的定义；见有关的SEP词条）。
然而分支类型论非常繁琐，而且在每个类型里面显然都能定义自然数，由此带来很麻烦的自然性问题。
罗素对此的回答——可化约性公理——背叛了逻辑主义的理想。
Quine证明了，对罗素的体系做well motivated的修正，就能得到ZFC，从而让罗素风格的逻辑主义看起来很不像真正的逻辑主义。

一个形式化的逻辑主义系统也是一个候选的形式主义的数学基础。
（这里强调“形式化”的逻辑主义系统是因为理论上可以有基于语义的逻辑主义系统，如罗素的数学原理的开头实际上并没有给出他用到的公理的完整列表，以至于他之后的学者要研究PM和公理化集合论的关系的时候需要自己做一遍形式化；应当注意这种做法实践上可能还真的没有更不严谨，因为即使是形式主义实际上也依赖于某些公认的语义上的东西。如果ZFC是自洽的，那么$\mathrm{ZFC}+\mathrm{con}(\mathrm{ZFC})$也是，但是这不代表我们能构造一个编码了对ZFC的不自洽的证明的自然数，因为后者是非标准的，可见形式主义需要事先假定某种关于初等算术的直觉）
与逻辑主义不同的地方是，（希尔伯特式的）形式主义的“支柱”并不是逻辑，而是某种很弱的有穷主义算术。
这是“形式主义将数学看成符号的操作”的更加substantial的意思：对抽象的（无穷的）数学对象的讨论可以看成对一些可以用某种有穷主义形式化的对象（字符串？）的操作。
我们今天知道希尔伯特计划不可能成功，因为一个很弱的有穷主义算术的自洽性无法在它里面证明。

如前所述，公理化集合论可以看成一个失败的逻辑主义尝试的副产品，并且符合形式主义的口味。
集合论自身并不endorse任何哲学观念。
一个使用集合论的数学家甚至可以是柏拉图主义者。

公理化集合论相较常规的数学实践其实仍嫌不够自然，如$1 \in 2$这样的命题对数学家来说非常奇怪。
不过，这些不自然之处都可以通过扩展语言来修改。
设想我们定义出自然数、有理数、实数、复数之后，建立从自然数到有理数、从有理数到实数等的嵌入，然后之后只在实数集中工作，并用原生自然数编码字符串，然后只讨论形如$(\texttt{type}, \texttt{value})$的ZFC对象（本身也是一个集合），并规定谓词在定义时需要附加类型，如果类型不匹配则自动输出“假”。
这样就得到了一个和日常工作中的数学语言一致的新语言。它有一个类型系统，但不是真正的类型论。
实际上这就是Mizar proof assistant的做法：它的“类型系统”的主要用处是自动化证明。
另一方面，罗素风格的逻辑主义和日常语言的差别是过于大了。

是否能拯救逻辑主义仍然是数学哲学领域的开放问题。

# 所有数学基础都以某种语义为起点

前面提了希尔伯特式的形式主义依赖于初等算数。arXiv 1807.05641这篇文章结尾的地方对此有一定的讨论。
一个“严格”的形式主义者的行为是让人无法理解的：他即使已经证明了不存在自然数$a, b$使得$a^2 = 2 b^2$，也不敢下断言说“你写程序找满足这种条件的数是找不到的”。
符号和对符号的manipulation都本质上是抽象的，和自然数一样抽象，承认前者不承认后者是办不到的。

因此形式主义者仍然要以对自然数、符号的语义的直觉为起点。

这就又带来了另一个结论。
元数学的讨论本身也必须在一个形式系统里面做，这个形式系统应该是什么呢？
比如说要论证皮亚诺算术的无矛盾性，这个论证要在什么系统里面做？
如果我们承认前述的一切数学基础都要以某种语义为起点的结论，那其实这个论证如果只依赖相较而言没什么争议性的性质，就是可以接受的。

所以我们应该要对皮亚诺算术的无矛盾性有一定的信心。
如果我们对此没有信心，那什么无矛盾性证明都不能让我们满意，因为用来证明一个公理系统无矛盾的公理系统本身可能是有矛盾的。

但这种信心又意味着实际上我们会需要不停地隐含地添加公理，因为“皮亚诺算术无矛盾”本身是一个关于自然数的命题，即$\mathrm{Con}(\mathrm{PA})$，所以如果我们接受皮亚诺算术无矛盾，那么就需要接受$\mathrm{Con}(\mathrm{PA})$的正确性，以及$\mathrm{Con}(\mathrm{PA} + \mathrm{Con}(\mathrm{PA}))$，等等。
自然数的结构其实是非常复杂的。
