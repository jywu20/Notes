The number of possible arrangements to put $k$ people into $n$ seats on a circle
in which there is at least one empty seat between two people is
\begin{equation}
    \frac{n}{n - k} \binom{n - k}{k},
\end{equation}
given that $2k \leq n$; otherwise no such arrangement is possible.
This is because of the follows.

First, we manually pick one people to be ``the first''.
Suppose $x_i$ is the number of seats after the $i$th person;
the $k$th person still needs to consume at least one seat,
because they need these seats as a separation from the first person.
So we have 
\begin{equation}
    \sum_{i = 1}^{k} x_i = n - k, 
\end{equation}
where for each $i$, $x_i \geq 1$.
The number of possible solutions can be seen as the number of ways to 
insert $k - 1$ walls into $n - k - 1$ spaces between $n - k$ objects,
and two walls can't appear in the same space.
So the number of solutions is $\binom{n - k - 1}{k - 1}$.
Now we need to pair these arrangements to the circle.
There are $n$ ways to place the first person.
However, note that in this process, we're permute the $x_i$'s as well,
causing double counting.
There are $k$ ways to permute $x_i$'s.
Therefore the final number of arrangements is 
\begin{equation}
    \binom{n - k - 1}{k - 1} \cdot \frac{n}{k} = \frac{n}{n - k} \binom{n - k}{k}.
\end{equation}
