# 理论框架

很多描写语法作者有一个奇怪的癖好，就是虽然他们嘴上说自己坚持的是functionalist和构式中心的语言观，不认同有任何意义上的普遍语法，等等，但是这些观点其实并不能够从他们写出来的语法中看到：
常见的描写语法的套路都是差不多的，先是phonology一章，然后是各种词性的介绍，然后是NP的结构，然后是动词变位，然后是clause structure（以及话题焦点啥的），然后是clause combining；构词法可能有专门一章也可能散落在名词和动词的形态学介绍中。
其实我觉得熟悉生成语法，尤其是Distributed Morphology的人，即使之前从来没读过一本参考语法，对这个套路也应该很熟悉：词性其实就是categorizer phrase，各种构词法其实就是比较低层次的一些phrase的语音实现，NP的结构就是DP的结构，动词变位就是CP-TP-vP的functional head的部分，clause structure就是CP-TP-vP的骨架和其上的各种specifier的关系。
对新奇的语言现象的描写是嵌入在这个套路里面的，而不是真的要重新发明一个新的套路。

所以就是说他们做语言描写其实是有套路的，这个套路基本上就是Dixon在Basic Linguistic Theory三卷中提到的那些东西。
虽然Dixon在BLT中怒喷生成语法但你还别说这个套路其实真的……挺生成的（尤其是挺Distributed Morphology的），虽然notation很不一样……
就structure building来说，Dixon反对严格的层级结构分析——他主张不要说[to [the [fat man]]]，而是要说to<sub>beneficiary </sub> [fat man]<sub>determined</sub>。
我不知道他有没有意识到“主语是TP的specifier”和“主语和有时态标记的动词之间有一个（相比动词和宾语的依存关系不是很紧密的）依存关系”这两句话是可以机械地相互转换的。
所以虽然他——以及很多别的人——主张只保留NP和clause两级constituent，这个实际上没有对自然语言的语法复杂性做出任何有意义的限制，只不过是换了一下notation而已。
再举一个例子：“生成语法强加名词和动词的区分”和“存在普遍的名词通常涉及的语法关系和动词通常涉及的语法关系”实际上是一回事，可是对描写语言学家来说，前者就要痛批而后者就可以接受。
Dixon还主张大家研究剥离了表层形态的the underlying structure of languages（就是to the fat man的后面一种标注方法），这个所谓的underlying structure又不完全是语义结构，除了生成语法中的所谓的syntax proper以外我都想不到这还能是什么。

Haspelmath伟大、光荣、正确地指出了Basic Linguistic Theory是一个universal grammar式的套路，一个framework这一事实；他转而希望有“framework-free grammatical theory”。
但这里的问题是其实Basic Linguistic Theory工作得确实可以（去翻一番language science press出的语法就知道了），所以大概也没有改变工作方式的必要……
（Gates 2021开篇时直截了当地指出basic linguistic theory实际上就是经过功能主义改革的结构主义，虽然他还是在重复生成语法不经过调查就假设语法范畴云云）

总之很多时候生成语法是该骂（叠屋架床、经验事实稍微变一变很多理论就完蛋，等等），但是有些人给我的感觉就是骂魔怔了；你要真的学习他们的方法论思想，最好去读他们对具体问题的具体分析而不是他们的口号。（我记得David Adger在他的博客里面还吐槽过这个事情；所以学习描写语言学还是拜师傅比较好……）

# 语言学科学转向的困难

导致生成语法被一些人反对的一个重要原因是，乔姆斯基的认知革命没有真的完成。
他想做的事情实际上可以用很清楚的语言概括出来：人能讲语言，必然是来自人脑的某些特殊之处，而这些特殊之处也意味着可以用统一的constrained metalanguage来描写所有自然语言，且这种元语言只能描写自然语言。
所以生成语法的“晦涩”是可以理解的，因为我们实际上是要用尽可能简单的primitive把自然语言的语法结构捕捉到，并且也只捕捉可能的自然语言的语法结构。

一些人（比如说Haspelmath）会说，it's all notation——完全错误。牛顿力学实际上也只是notation而已，我们从来没有一个完全基于观测数据的“力”的定义。一个理论框架的合理性是通过它确实能描写观测数据得到辩护的，而不是按照Haspelmath所谓theory without framework来辩护：后者就好像一个物理学家指示他的学生“要观察”一样——没有一个理论框架，你甚至不知道应该观察什么的。

然而不幸，人类语言到底是复杂的，而现代的心理语言学和神经语言学充斥着模糊不清的数据和不可重复的成果。
理想情况下，理论语言学家应该能同时出现在医院里（分析一个不幸中了风而说不清话的老太爷到底是如何中风了）和田野上（观察哪些模式在自然语言中不会出现）。
可是实际情况是，两者之间的隔膜一直没有很好地打通，而两个领域都不甚可靠。

所以前述的描写语法作者一口咬定自己坚持的是functionalist和构式中心的语言观也就不奇怪了。
然而这里的问题在于，功能或者认知学派并不能否认乔姆斯基的这个构想：如果语言来自人脑的功能，那为什么不能使用统一的元语言描写所有自然语言？
他们能做的只有论证说“实际上我们应该用统一的元语言描写所有文化现象，而不只是语言”，而既然人的文化现象可以非常复杂，现阶段自然语言的描写不应该预设框架。
然而实际情况就是，我们只有关于possible language的理论（普遍语法，或者Basic Linguistic Theory），我们没有关于可能的文化的理论。
这也就解释了描写语法作者口嫌体正直的原因：所谓“语言可以reduce到domain general的文化上面”完全就是lip service。

# Usage based approach和凑字数学派

由于前述的“我们没有文化的普遍语法”的原因，近年来非生成语法的语言学有一个趋势，就是不自称constructivists或者functionalists，而是自称usage-based。
这有点像发在Language上的文章和发在Linguistic Inquiry上的文章的区别：前者的基调听着很生成，并且往往会用一些生成的黑话，比如说DP，CP之类，但是不会特别强调一定要把句子的整个derivation做出来，而后者是会花时间在很细枝末节的东西上面的。
然而usage-based的文章我觉得很多时候都趣味寥寥：它们很像拙劣的心理语言学论文，充斥着本身就不靠谱的统计推断，以及更加不靠谱的对这些统计推断的解读。
Tomasello等人会说这是beyond universal grammar, a unified framework of xxx——我觉得说unified sloppiness可能更加正确。

一个很简单的例子：小朋友学习语言的时候会吐出很多“成句”，比如说give me。
Usage based学派的人会正确地指出这一点，然后做一堆频率分析来指出这些成句出现得是如此频繁。
然后他们得出结论：小朋友一开始在模仿give me这些说法，而不知道其结构为何。
因此语法结构是后起的，可能是文化的趋同演化云云。

语法结构是不是文化的趋同演化呢？从纯观测的角度，它当然是：可是这是毫无意义的同义反复。
为什么儿童不学着猴子排列组合特定的叫声来交流？
人们喜欢说蝙蝠的翅膀和鸟的翅膀是趋同进化的产物：正确，然而它们能趋同进化这件事本身是由这两类生物都有能扑腾的手臂、五指的底层结构保障来支持的。
趋同进化能发生，这本身就意味着一些事情；另一方面地球生物还没有一个长得像飞机的。

回到give me的问题上。give me比give somebody出现得更频繁能说明什么呢？当然，有chunking，可是我们马上要讨论chunking是怎么来的。
完全有可能，give me出现得更加频繁是因为conversation participant更常出现在祈使句里面。
这样一来give me出现得更加频繁就不能支持“儿童学习且只学习完整的chunk”了。

其实要支持儿童学习chunk不难，看看历史语言学上被语法化的东西也就知道chunking是真实存在的。
然而生成语法和usage based的主要区别在于前者相信一个chunk在被学到以后会很快地被儿童加以分析，并且是以类似于成人的方式分析。
一个chunk被分析不代表类似结构的utterance都会被说出来：如前所述，有些utterance就是更频繁地被说出来，可是这个可能是语境决定的，而不能体现说话人内心语法的结构。
据说爱因斯坦说话非常迟，这大概并没有说明他的内心没有一部德语语法。

对儿童语言是否有语法结构的分析必须要建立在和成人的语言产出相对比的基础上。
如果两者的统计分布类似，那么儿童很可能一开始就在内心构建自己的语法了。
实际情况似乎确实是这样 (Yang 2013)。

Usage based人可以继续论证language use不是hierarchical的。他们也的确这样论证了。
然而现在，构建一个不使用语法结构的算法产生和人的语言产出的统计分布类似的分布的任务就落在他们肩上了。
（Yang 2013使用了一个简单的这种类型的算法，并证明它的产出和人的utterance不像）。

很不幸，usage based语言学似乎变成了一个凑字数学派。
这个学派最顶上的从业者在分析真正的数据并且有很多insight，可是他们发现的东西不使用usage based的理论也能被叙述。
这又体现出了此学派的另一个问题，就是extreme lack of theorization：
我们既不知道什么东西能证实usage based approach，也不知道什么东西能证伪它。
usage based的文章往往是一堆被oversell的数据的大杂烩，而缺少一个完整的故事，就好比一节只教授如何作图和拟合数据的物理课一样。
而这个学科不甚高明的从业者则和低端心理学从业者没有区别了。

缺乏完整故事导致usage based approach的“理论创新”往往围绕着一个模糊不清的概念进行，比如说“语法化”——这个概念似乎完全落在reanalysis的概念里面。
Campbell在他的著名的历史语言学教科书里面批评过说语音弱化是语法化、语义漂白是语法化、这个是语法化那个是语法化的糊成一团的写作风格：他明确指出所有这些现象都有更加恰当和明确的传统机制可以解释。

不幸，这并没有妨碍他们写出一篇又一篇的review来论证the usage based approach is thriving。

# Mood

我们经常说不要局限在拉丁语范式中，但这并不能帮助我们辨认哪些东西是真的只属于拉丁语的。
拉丁语语法中的mood这个说法实际上是modality和陈述、疑问、祈使等CGEL中所谓的clause type这两个语法范畴的杂糅：
indicative = realis declarative, subjunctive = irrealis declarative, 
imperative = (no modality) imperative。
这个杂糅在描写拉丁语语法时是好用的，其他时候则未必好用。

# 日语的学校文法和教育文法之争

英文维基上的做法是，将“未然、已然”这些标签看成verb base——此处base一词指的不是一个具体的、成分树中比词更小的成分，而是前者的尾部形式，只有有限种可能性——而将“现在-肯定-简体”这样的完整的动词词尾看成inflection form。
这就是非常合理的做法：助动词的变形本身不携带任何抽象的语法信息：这就是很普通的词素随着形态句法环境的变化发生机械的变化而已；它是morphophonological的而不是morphosyntactic的（它们的复杂性和“一个”中的“一”要读第二声这种没啥大的区别）。
这样一方面保留了学校文法细致划分词尾、不需要去把每种活用形都记下来的好处，一方面和普通语言学完全接轨：
比如说你可以拿这套体系去分析任何一种词素有稳定的contextual allomorphs的语言。
真的，好像日本人讲韩语就是这么讲的——见[此处](http://www.tufs.ac.jp/ts/personal/choes/korean/base/Ebase.html)。
可惜的是好像很多中文教材都不太愿意在一开始就点透这层窗户纸……
