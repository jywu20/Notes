\section{Introduction to iTensor}

ITensor is a package providing utilities to quickly building DMRG and other tensor network simulation programs \cite{itensor}.

Each \texttt{SiteSet} provides:
\begin{itemize}
    \item A set of available operators defined on each set. To add custom operators you need to create your own \texttt{SiteSet}.
\end{itemize}
