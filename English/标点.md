# English punctuations
| Name                        | Symbol                                  |
| :-------------------------- | :-------------------------------------- |
| period, full stop           | .                                       |
| comma                       | ,                                       |
| colon                       | :                                       |
| semicolon                   | ;                                       |
| exclamation mark            | !                                       |
| hyphen                      | -                                       |
| asterisk                    | *                                       |
| apostrophe                  | ', as appears in "the girl's" or aren't |
| dash                        | --                                      |
| underscore                  | _                                       |
| single quotation mark       | '                                       |
| double quotation mark       | "                                       |
| parenthesis, round brackets | ()                                      |
| square brackets             | []                                      |
| angle brackets              | <>                                      |
| curly brackets              | {}                                      |
